# CHAPTER 4 – COMMUNICATION (TRAO ĐỔI THÔNG TIN)

Các service trong hệ thống giao tiếp với nhau thông qua HTTP REST API và sử dụng định dạng dữ liệu JSON.

Ví dụ:

- Auth Service gửi request đến User Service để lấy thông tin người dùng theo email.

- User Service phản hồi dữ liệu để phục vụ quá trình xác thực.

Giao tiếp giữa các service được kiểm thử bằng công cụ curl và hoạt động ổn định.

# CHAPTER 7 – BACKUP & DATA MANAGEMENT (SAO LƯU & QUẢN LÝ DỮ LIỆU)

User Service sử dụng H2 in-memory database trong quá trình phát triển và kiểm thử.

## Hibernate được sử dụng để:

- Tự động tạo bảng

- Quản lý entity

- Đồng bộ dữ liệu với database

Dữ liệu được kiểm tra thông qua H2 Console và các câu lệnh SQL. Trong môi trường thực tế, H2 có thể được thay thế bằng các hệ quản trị cơ sở dữ liệu như PostgreSQL hoặc MySQL.
